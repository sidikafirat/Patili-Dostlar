# Patili-Dostlar
VSC compilerını kullanarak yazdığım JS oyun projemdir. Bu projemi geliştirirken udemy, youtube gibi öğrenim kanallarından yararlandım. Aşağıdaki Linkten oyuna girerek küçük çocuklara hayvan sevgisini aşılamak isteyen herkes bu oyunu oynayabilir.
# Nasıl Oynanır?
Başla butonun tıkladıktan sonra sağ-sol, yukarı-aşağı komutları ve iki farklı input ile de oynanabilir. Amaç sepeti hareket ettirerek yukarıdan random gelen mamaları toplamaktır.
